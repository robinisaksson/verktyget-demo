verktyget

verktyget - "the tool" is a bunch of useful es6 javascript modules. this demo site shows how you can use the different modules. i have tried to use as much of the functionality as possible without creating to much complexity. you can always contact me with your questions.

overview

these are the classes or modules that you can use in your project.


  • event dispatcher
  • dom helpers
  • device info
  • image loader
  • video loader
  • video player
  • jsonp loading
  • ajax loading
  • url helper
  • scroll detection
